ゲーム名：ふらふらぺんぎん
制作：チーム制作
チーム構成：　Pl:1、Pr:1、Gr:1
担当：プログラム全般
使用言語：c#
動作環境：AndroidOS搭載のタブレット（推奨：Nexus9）
制作時間：２０時間


-----説明-----
Unityで制作しました。
子供のペンギンを親ペンギンの元に連れて行くゲームです。
海を渡るために、氷を置いていって子供ペンギンの道を作ってあげます。
海に落ちるとゲームオーバーです。
氷は画面をタッチして置くことができます。
タッチ長押しすることで氷を大きくすることができます。
ただし氷の上に長く居続けると氷が割れて海に落ちてしまうので、大きく作り過ぎるのも危険です。

-----見てほしいところ-----
ペンギンの動作にタブレットの傾きの値を取り入れるように設計
氷の上に載っている判定を、氷のあたり判定と氷にあたっている時間の両方で行い、正確に判定が出るように工夫しました。
氷を一つのオブジェクトを複製して作っていくので、グラフィックのデータなどが最小限に抑えられるように設計。
40時間で作るという課題のもと制作したので、コードをできるだけ短く書いて早く作るように意識しまいした。

-----制作して感じたこと-----
40時間で作るということを意識して作っていたので、コードを出来るだけ簡単な処理になるように考えて書きました。締切りを意識して制作していくと20時間でつくることができ、余裕を持って制作することが出来ました。
この制作を通して話し合いの大切さを感じることが出来ました。短い期間なので話し合いを多く行い、チーム内でゲームのイメージと各々の考えを共有し制作したので、短時間で作れたのだと思います。この経験を活かし、ホウレンソウの徹底と自分の考えを他人にも分かるように伝えることを意識していきたいと思います。

-----操作方法-----
指一本で遊べます。

タブレットの傾きを検知して傾きに合わせて
子供ペンギンがフラフラするので氷の上から落ちないよう注意してください
